# Predicción de Resistencia del Concreto

Sistema desarrollado con Machine Learning para estimar                              la resistencia del concreto (MPa) a partir de las propiedades de la mezcla.

# Tecnologías utilizadas
- Python
- Streamlit
- Scikit-learn
- Pandas
- Machine Learning

# Funcionalidades
- Predicción automática de resistencia
- Carga de archivos Excel
- Visualización gráfica
- Modelo Random Forest Regressor

# Métricas del modelo
- R² = 0.87
- RMSE = 5.66
